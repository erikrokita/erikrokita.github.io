  • Extra page 6 is an early addition of a music game. Mostly nonfunctional, but the foundations are mostly in place. The level notes are defined under javascript variables in their own javascript files and each character is read from left to right. A 0 represents no notes, a 1 represents a single-hit note. This allows to easily create new note files for other songs with a constant bpm. Bpm and songname are determined in the html tags, but are planned to also be migrated to the javascript file as well. In the console, you can see the information of the notes being read when you press enter on the webpage
